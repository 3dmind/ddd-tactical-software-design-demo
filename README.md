# Domain-Driven Design: Tactical Design Demo

## Description

Showing tactical design patterns inside a [Nest](https://github.com/nestjs/nest) application.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
